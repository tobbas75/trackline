### Footprint Compatibility

The package is **0.1 mm narrower** (3.75 vs 3.85 mm). This is within typical pad tolerance for hand-assembly, but:

- If pads were sized exactly to 3.85 mm, the A01 will be slightly loose — may need pad size adjustment
- If using stencil-based reflow, the 0.1 mm difference should self-centre during reflow
- **Recommendation:** Design new footprint to 3.75 mm spec. Do not assume old SFH4725S footprint is compatible without checking.

### SFH 4718A Warning

**SFH 4718A is 850 nm — WRONG wavelength for covert operation.** 850 nm LEDs produce visible red glow that animals (and humans) can see. The config.h file incorrectly references SFH4718A in LED module ID comments — this needs correction if 940 nm standard modules are intended.

### Thermal Analysis (per LED)

At 175 mA, ~2.4V Vf (estimated):
```
P_electrical = 175 mA x 2.4V = 420 mW per LED
P_optical = (175/1000) * (1340/1000) * 1000 = ~234 mW (scaled linearly — approximate)
P_thermal = 420 - 234 = ~186 mW per LED (dissipated as heat)

Tj_rise = P_thermal x Rth_JS = 0.186W x 6.6 K/W = 1.2C above solder point
```

At 30 ms strobed duty, thermal rise is negligible. Even worst-case stuck-on failure:
```
Tj_rise_continuous = 0.186W x 6.6 = 1.2C (still safe — AL8861 thermal shutdown protects)
```

**Thermal margin is excellent for this application.**

## Application Notes for This Design

1. **Parallel topology.** 8 LEDs in parallel, NOT series. Series string of 8 LEDs would require ~21.6V (8 x 2.7V) — impossible from 5V VLED buck. Each LED gets its own 1 ohm ballast resistor for current sharing.

2. **940 nm is covert.** Peak wavelength 950 nm produces no visible glow to human eyes or most mammal eyes. This is essential for wildlife monitoring (animals are not startled) and anti-poaching (camera location not revealed).

3. **80 deg beam angle.** The 40 deg half-angle should be matched to the camera's field of view (~100 deg HFOV with M12 lens). Two LEDs may need slight angling for edge coverage. The 940nm Long Range module uses narrower-beam variants for distance.

4. **Strobe synchronisation.** TIM1 CH1 on STM32N6 generates STROBE signal gated to MIPI CSI-2 frame-valid. The AL8861 VSET pin responds in ~100 us (fast enough for 30 ms pulse).

5. **LED board is field-swappable.** Connected via JST GH 4-pin. No soldering for module changes. Module auto-detected by ID resistor at boot.

## Known Issues

1. **Footprint mismatch with SFH4725S.** 3.75 mm vs 3.85 mm. Design new footprint, do not reuse old SFH4725S pads without verifying.

2. **Vf at 175mA is estimated.** The datasheet specifies Vf at 1A (2.65V typ) and 5A pulsed (3.1V typ). The operating point of 175 mA per LED is well below the characterised range. Vf at 175 mA is likely ~2.3-2.5V but must be verified from the datasheet I-V characteristic curve or by bench measurement of samples.

3. **Parallel LED current sharing.** Without ballast resistors, Vf mismatch between LEDs causes uneven current distribution. The hottest LED runs away thermally. The 1 ohm ballast per LED provides ~175 mV drop at 175 mA, which is adequate for current sharing given typical Vf variation of ~50 mV between binned LEDs.

4. **config.h LED reference inconsistency.** config.h LED_ID_940NM_STANDARD references "SFH4718A 60 deg" — but SFH4718A is an 850 nm part. The actual 940 nm standard module uses SFH 4725AS A01 at 80 deg. This comment needs correction in config.h.

5. **AEC-Q102 qualification.** The A01 variant is automotive-qualified. Good for reliability in harsh outdoor conditions (temperature cycling, humidity), but also means the part may be allocated to automotive customers during shortages.
