# Molex 2066400001 — Active GNSS Patch Antenna

## Summary

Active GNSS ceramic patch antenna with integrated 2-stage LNA and SAW filter. 25 x 25 x 6.5 mm form factor with 60mm coaxial cable and U.FL connector. Supports GPS L1, GLONASS, BeiDou, and Galileo.

## LNA Power Supply

| Parameter | Value | Notes |
|-----------|-------|-------|
| Supply Voltage | 3.0 +/- 0.3 V | Fed via coax from GPS module VCC_RF pin |
| DC Current | 8.5 +/- 3 mA | At 3.0V supply |
| Power Source | MAX-M10S V_RF pin | DC bias on coaxial cable |

The LNA is powered via the RF coaxial cable (DC bias injected by the GNSS module). The MAX-M10S provides VCC_RF output on its antenna pin for this purpose.

## CRITICAL: Active Antenna + MAX-M10S LNA Configuration

**This is an active antenna with a 28 dB integrated LNA.** The MAX-M10S has its own internal LNA which is enabled by default. Using both LNAs in cascade will likely cause signal saturation and degraded performance.

**Required configuration change:**
- Set `CFG-HW-RF_LNA_MODE` to bypass/disable the MAX-M10S internal LNA
- This is configured via UBX protocol over I2C/UART during GPS initialisation
- Without this change, total gain would be ~50+ dB which will saturate the ADC front end

**Firmware action:** Add LNA mode configuration to the GPS initialisation sequence in the HAL. The u-blox documentation covers this in the MAX-M10S integration manual.

## Application Notes for This Design

1. **Mounting:** Adhesive mount inside the enclosure lid (top face). The polycarbonate enclosure is RF-transparent at L-band frequencies. Orient the patch face upward (toward sky) for best satellite visibility.

2. **Ground plane:** The antenna has an integrated ground plane within the 25x25mm module. No additional ground plane is needed, but performance improves with a larger ground plane underneath. Mounting on or near the main PCB copper pour is beneficial.

3. **Keep-out zone:** Avoid placing metal components, wires, or battery cells directly above the patch element. Minimum 10mm clearance above the patch face.

4. **Cable routing:** 60mm cable is short. Route directly from the antenna to the J_GPS u.FL connector on the main board. Avoid sharp bends. If 60mm is insufficient for the enclosure layout, a u.FL extension pigtail may be needed.

5. **Connector:** U.FL mates with J_GPS (LCSC C88374) on the main board.

6. **V_BCKP supercapacitor:** The 0.47F supercap on V_BCKP provides hot-start capability (~7 hours). The antenna module itself has no backup requirement — it is always powered via VCC_RF when the GPS module is active.

## Comparison with Passive Antenna Alternative

| Parameter | Active (2066400001) | Passive patch | Notes |
|-----------|---------------------|---------------|-------|
| Sensitivity | Better (28 dB LNA) | Worse | Active antenna compensates for cable loss |
| TTFF | Faster | Slower | Better signal = faster acquisition |
| Power | +8.5 mA when GPS active | 0 mA | GPS is power-gated, only on during fix |
| Complexity | LNA mode config needed | Simpler | Must configure MAX-M10S to bypass internal LNA |
| Cost | ~$3.00 AUD | ~$1.50 AUD | |
| Size | 25x25x6.5mm | Similar | |

For a camera trap that needs fast GPS fixes to minimise wake time and battery drain, the active antenna is the better choice despite the LNA configuration requirement.
