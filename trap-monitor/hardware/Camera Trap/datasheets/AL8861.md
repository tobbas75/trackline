## Current Sensing Circuit

The AL8861 uses **high-side current sensing**. An external sense resistor connects between VIN and the CS/ISENSE pin.

```
    VLED (5V) ───┬──── Pin 1 (VIN)
                 │
                R_sense (0.18 ohm)
                 │
                 └──── Pin 5 (CS/ISENSE)
```

The nominal output current is:

```
I_LED = V_sense / R_sense = 0.25V / R_sense
```

| R_sense | I_LED | Application |
|---------|-------|-------------|
| 0.18 ohm | 1.39A (~1.4A) | **This design — 8 parallel LEDs at 175mA each** |
| 0.25 ohm | 1.00A | Single-driver reduced-current option |
| 0.36 ohm | 0.69A (~700mA) | Split dual-driver option (4 LEDs each) |

**R_sense power dissipation:** P = I^2 x R = 1.4^2 x 0.18 = 0.35W. Use 0402 or 0603 rated for 0.5W minimum. Consider 0805 for thermal margin.

## VSET Pin Behaviour

The VSET pin controls LED brightness and standby mode:

```
VSET voltage:
  < 0.2V  → Standby (output OFF, < 1 uA quiescent)
  0.2-0.5V → Transition region (undefined output)
  > 0.5V  → Active (output starts, LED current flows)
  0.5-2.5V → Analog dimming (linear, proportional to VSET)
  > 2.5V  → Full brightness (I_LED = 0.25V / R_sense)
```

**In this design:** STROBE signal is 3.3V logic from STM32N6 TIM1 CH1.
- STROBE LOW (0V) = VSET < 0.2V = standby = LEDs OFF
- STROBE HIGH (3.3V) = VSET > 2.5V = full brightness = 1.4A through LEDs

### Soft-Start

With no external capacitor on VSET:
- Default soft-start time: **~0.1 ms**
- This is 0.3% of the 30 ms strobe pulse — acceptable, no visible ramp

If slower ramp is needed: add C_ext from VSET to GND:
```
t_softstart = C_ext x 1.5 ms/nF
```
Example: 10 nF → 15 ms soft-start (would consume half the strobe pulse — too slow).

**Recommendation:** No external cap on VSET. 0.1 ms start is fine for 30 ms pulses.

### Enable Response Time

The AL8861 responds to VSET transitions within ~100 us (VERIFY from datasheet). This means:
- STROBE HIGH → LEDs on within ~100 us
- STROBE LOW → LEDs off within ~100 us (inductor current decays)
- Total latency is small relative to 30 ms exposure window

## LED Topology — CRITICAL

**LEDs must be in PARALLEL, not series.**

```
WRONG (series — IMPOSSIBLE from 5V):
  VLED 5V → AL8861 → L → LED1 → LED2 → ... → LED8 → GND
  Total Vf = 8 x 2.7V = 21.6V >> 5V VLED  ✗ IMPOSSIBLE

CORRECT (parallel with ballast resistors):
  VLED 5V → R_sense → AL8861 → L → ┬─ R_ballast ─ LED1 ─┬─ GND
                                     ├─ R_ballast ─ LED2 ─┤
                                     ├─ R_ballast ─ LED3 ─┤
                                     ├─ ... (x8)         ─┤
                                     └─ R_ballast ─ LED8 ─┘
```

The AL8861 bucks 5V down to ~2.7V (Vf + ballast drop) and drives all 8 LEDs in parallel. Total current 1.4A is shared ~175 mA per LED.

### Ballast Resistors

**1 ohm per LED** for current sharing. Without ballast, LEDs with lower Vf hog current and overheat.

```
V_ballast = 175 mA x 1 ohm = 175 mV
P_ballast = 175 mA x 175 mV = 30.6 mW per resistor (0402 is fine)
```

The 175 mV ballast drop provides adequate current sharing given typical Vf variation of ~50 mV between binned LEDs.

## Power

| State | Input Current (from 5V VLED) | Notes |
|-------|------------------------------|-------|
| Standby (VSET < 0.2V) | < 1 uA | VLED rail can be power-gated (PB6) for zero quiescent |
| Active (1.4A LED current) | ~1.4A + driver overhead | Efficiency ~85-90% at this operating point — VERIFY |
| VLED rail gated OFF | 0 | PB6 LOW disables AP63300 U15 entirely |

**In sleep:** The 5V VLED buck (U15, AP63300) is power-gated by PB6 = LOW. Zero current through AL8861.

**During burst:** PB6 HIGH enables U15. STROBE pulses drive VSET. 300 ms total on-time per burst at 1.4A.

## Application Notes for This Design

1. **Input voltage.** 5V from VLED rail (AP63300 U15). Well within 4.5-40V range.

2. **STROBE connection.** VSET pin directly connected to STROBE signal (3.3V logic, TIM1 CH1 PA8 on N6). No level shifting needed — 3.3V > 2.5V full-brightness threshold.

3. **Inductor selection.** Hysteretic buck needs inductor. Typical 10-33 uH, rated for 2A+. VERIFY recommended inductor value from datasheet application circuit. Wurth evaluation board design is a good reference.

4. **Output capacitor.** Usually not required for constant-current LED driver (LEDs are the load). Small ceramic (~1 uF) at LED node for EMI filtering is optional.

5. **Thermal dissipation.** SOT89-5 tab connects to GND pad. Use generous copper pour under the part on the LED daughter board. At 1.4A with ~90% efficiency, ~0.4W dissipated in the driver IC. SOT89-5 can handle this with adequate copper.

6. **Schottky diode.** The AL8861 has an integrated power FET but may need an external Schottky diode for the buck topology. VERIFY from datasheet — some hysteretic drivers are fully integrated, others need external diode.

7. **Fail-safe.** If STROBE stuck HIGH, the AL8861 thermal shutdown (~165C) protects the LEDs and driver from damage. The 5V VLED rail is also current-limited by the AP63300 (3A max).

## Known Issues

1. **TSOT-25 vs SOT89-5 current rating.** The most critical procurement issue. TSOT-25 (AL8861Q) is 1.0A max — insufficient for 1.4A load. SOT89-5 (AL8861Y-13) provides 1.5A. This was caught in the V1.2 schematic review and is documented in LED_MODULE_SPEC.md and SCHEMATIC_CHANGES_V1.2.md.

2. **Series vs parallel LED topology.** Series string of 8 LEDs requires ~21.6V — impossible from 5V VLED. LEDs MUST be in parallel with ballast resistors. Also caught in V1.2 review.

3. **Hysteretic switching frequency.** The AL8861 uses hysteretic control, so switching frequency varies with VIN, VLED, inductor value, and LED Vf. Nominally ~1 MHz but expect variation. This can cause EMI — place driver and inductor close together on the LED daughter board with short, wide traces.

4. **VSET pin floating.** If the STROBE trace is disconnected (cable fault), VSET could float. The internal bias may pull it above 0.5V, causing LEDs to turn on unexpectedly. Consider adding a weak pull-down (100 kohm) from VSET to GND on the LED daughter board as a fail-safe.

5. **Pinout verification.** The SOT89-5 pinout above is from training data and MUST be verified against the DS39435 datasheet before PCB layout. Getting VIN and LX swapped would short the input supply through the inductor.

6. **Alternative topology.** If single-driver reliability is a concern, an alternative is two AL8861QNT7G (TSOT-25, 1.0A each), each driving 4 parallel LEDs at 700 mA (R_sense = 0.36 ohm). This provides redundancy — if one driver fails, half the LEDs still work. Trade-off: more board area, higher BOM cost.
