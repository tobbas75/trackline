**2. ImageViewer.tsx** (VIEW-02, VIEW-03)

Props interface:
```typescript
export interface ImageViewerProps {
  event: CameraEvent;
  imageUrl: string;           // pre-resolved Supabase Storage public URL
  onClose: () => void;
}
```

Implementation:
- Modal overlay: fixed inset-0 z-[1200] with semi-transparent backdrop `bg-black/70`.
- Close button top-right: "X" button with `text-white` on the overlay.
- Click backdrop to close (not the image container).
- Image container: `relative inline-block` centred in the modal. Use `<img>` tag with `max-h-[85vh] max-w-[95vw] object-contain rounded-lg`.
- Bounding box overlays: For each detection in `event.detections`, render an absolute-positioned `<div>` over the image:
  ```
  style={{
    position: 'absolute',
    left: `${detection.x * 100}%`,
    top: `${detection.y * 100}%`,
    width: `${detection.width * 100}%`,
    height: `${detection.height * 100}%`,
  }}
  ```
  - Border: `border-2 rounded-sm` with colour based on confidence:
    - >= 80%: `border-green-400`
    - >= 50%: `border-yellow-400`
    - < 50%: `border-red-400`
  - Label: positioned above the box (negative top offset), showing `{class_name} {Math.round(confidence * 100)}%` in a small pill: `bg-black/75 text-white text-xs px-1.5 py-0.5 rounded whitespace-nowrap`
- If no detections, show "No detections" text below the image.
- Event metadata below image: captured_at formatted, model_name, battery_percent if available. Style: `text-sm text-white/80`.
- Escape key closes modal (useEffect with keydown listener).
