**TypeScript interfaces for NE301 payload** (define at bottom of file, same pattern as ingest-sms):
```typescript
interface NE301Payload {
  metadata: { timestamp: number; width: number; height: number };
  device_info: { battery_percent: number; communication_type: string };
  ai_result: {
    model_name: string;
    inference_time_ms: number;
    ai_result: {
      type_name: string;
      detection_count: number;
      detections: Array<{

        x: number; y: number;
        width: number; height: number;
      }>;
    };
  };
  image_data: string;
}
```

NOTE: The NE301 payload has a nested `ai_result.ai_result` — this is NOT a typo. The outer `ai_result` contains model metadata, the inner `ai_result` contains the actual detection results. Match this exactly.

**Payload validation function** `validateNE301Payload(body: unknown): { valid: true; payload: NE301Payload } | { valid: false; reason: string }`:
- Check body is object, not null
- Check `metadata` exists with `timestamp` (number), `width` (number), `height` (number)
- Check `device_info` exists with `battery_percent` (number), `communication_type` (string)
- Check `ai_result` exists with `model_name` (string), `inference_time_ms` (number)
- Check `ai_result.ai_result` exists with `detection_count` (number), `detections` (array)
- Check each detection has `class_name` (string), `confidence` (number 0-1), `x`, `y`, `width`, `height` (numbers 0-1)
- Check `image_data` exists and starts with `"data:image/jpeg;base64,"` or `"data:image/png;base64,"`
- Return `{ valid: false, reason: "descriptive message" }` for each failure (per MQTT-06)

**Extract device_id from request** — MQTT broker webhook will include topic or device_id. Support two patterns:
1. Query parameter: `?device_id=UNIT_CAM_001` or `?topic=ne301/UNIT_CAM_001/upload/report`
2. JSON field: add optional `device_id?: string` and `topic?: string` to payload interface

Parse device_id:
```typescript
function extractDeviceId(url: URL, body: Record<string, unknown>): string | null {
  // Direct device_id param or body field
  const directId = url.searchParams.get("device_id") || (body.device_id as string);
  if (directId) return directId;

  // Extract from MQTT topic: ne301/{device_id}/upload/report
  const topic = url.searchParams.get("topic") || (body.topic as string);
  if (topic) {
    const parts = topic.split("/");
    if (parts.length >= 2) return parts[1];
  }

  return null;
}
```

**Image storage** (per IMG-02, IMG-03):
```typescript
// Strip data URI prefix to get raw base64
const base64Prefix = "data:image/jpeg;base64,";
const base64Data = payload.image_data.startsWith(base64Prefix)
  ? payload.image_data.slice(base64Prefix.length)
  : payload.image_data.slice(payload.image_data.indexOf(",") + 1);

// Decode base64 to Uint8Array
const binaryStr = atob(base64Data);
const bytes = new Uint8Array(binaryStr.length);
for (let i = 0; i < binaryStr.length; i++) {
  bytes[i] = binaryStr.charCodeAt(i);
}

// Upload to Supabase Storage: {org_id}/{unit_id}/{event_id}.jpg
const storagePath = `${unit.org_id}/${deviceId}/${cameraEvent.id}.jpg`;
const { error: uploadError } = await supabase.storage
  .from("camera-images")
  .upload(storagePath, bytes, {
    contentType: "image/jpeg",
    upsert: false,
  });

if (uploadError) {
  console.error("[INGEST-MQTT] Storage upload error:", uploadError);
  // Continue — event is stored, image upload failure is non-fatal
} else {
  // Update camera_event with image_path
  await supabase
    .from("camera_events")
    .update({ image_path: storagePath })
    .eq("id", cameraEvent.id);
}
```

**Detection inserts** (per CAM-05) — look up species_id from species table, insert each detection:
```typescript
const detections = payload.ai_result.ai_result.detections;
if (detections.length > 0) {
  // Batch lookup species by class_name
  const classNames = [...new Set(detections.map(d => d.class_name.toLowerCase()))];
  const { data: speciesRows } = await supabase
    .from("species")
    .select("id, name")
    .in("name", classNames);

  const speciesMap = new Map(
    (speciesRows || []).map(s => [s.name, s.id])
  );

  const detectionRows = detections.map(d => ({
    camera_event_id: cameraEvent.id,
    species_id: speciesMap.get(d.class_name.toLowerCase()) || null,
    class_name: d.class_name,
    confidence: d.confidence,
    x: d.x,
    y: d.y,
    width: d.width,
    height: d.height,
  }));

  const { error: detError } = await supabase
    .from("camera_detections")
    .insert(detectionRows);

  if (detError) {
    console.error("[INGEST-MQTT] Detection insert error:", detError);
  }
}
```
